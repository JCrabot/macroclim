These files are used to run the MARXAN software (Ball et al. 2009).

The "input.dat" file contains the indications for the execution command and 
the "Marxan_x64.exe" is the execution command of MARXAN. They are called directly
from R in the script "8.Plot_MARXAN_priority_maps.R", in the chunk "Run
MARXAN"."

Ball, I., Possingham, H. P. & Watts, M. Marxan and relatives: Software for
spatial conservation prioritization - UQ eSpace. in Spatial conservation
prioritisation: Quantitative methods and computational tools vol. Chapter 14
Pages 185-195 (2009).
